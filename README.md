# 📝 Resume Builder - React Project

A modern, responsive **Resume Builder** web application built with **React.js**, **HTML5**, and **CSS3**. This project allows users to create, preview, and customize their resumes live in the browser — no design skills needed.

---

## 🚀 Features

- ⚡ Live resume preview as you type
- 🎨 Clean and responsive UI
- ✏️ Editable sections: Personal Info, Skills, Projects, Experience, Education
- 📄 Download-ready layout (for PDF export via browser print)
- 📱 Mobile-friendly design

---

## 🛠️ Tech Stack

| Tech        | Use Case                         |
|-------------|----------------------------------|
| React.js    | Component-based UI structure     |
| HTML5       | Semantic structure               |
| CSS3        | Styling and layout               |
| React Hooks | State management and interactivity |
